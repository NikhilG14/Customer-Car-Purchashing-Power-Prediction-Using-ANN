# Customer-Car-Purchashing-Power-Prediction-Using-ANN
The intention of this project is to predict the highest amount a customer is willing to spend when purchasing a new car using ANN. 

Features:
  Customer Name	
  Customer e-mail	
  Country	
  Gender	
  Age	
  Annual Salary	
  Credit Card Debt	
  Net Worth
  
Output:
  Car Purchase Amount
